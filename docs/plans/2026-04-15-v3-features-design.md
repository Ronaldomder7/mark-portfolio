## 3. Hero 手电筒彩蛋

**视觉机制：**
- 默认 Hero 显示 3 行："人不能活一辈子。/ 最后只有两个字—— / 严父，或者慈母。"
- 角落出现 🔦 图标，可拖动
- 用户拖动 → 光斑扫过 3 行文字，扫过区域的文字"反转成黑块"（黑洞感），露出底下更大的 2 行字
- 揭示内容：**"你想要 / 怎样活这一生？"**（2 行，字号放大到填满原 3 行的视觉面积）
- 完全扫过 → 大字完整呈现 3 秒 → 淡出恢复原 3 行
- 彩蛋触发过后 🔦 图标淡出（一次性体验）

**技术实现：**
- SVG mask + CSS blend mode
- 手电筒是圆形光斑（50-80px 半径），跟随鼠标/触摸
- 光斑内：表层文字透明度降到 0，同时露出底层的大字
- 用 `mask-image: radial-gradient(...)` 实现圆形揭示
- 拖动进度追踪：记录已扫过的区域占比，达到 80%+ 触发"完整呈现→淡出"动画

## 5. 足迹地图手势交互

### 摄像头模式（授权允许后）

**技术栈：** MediaPipe Hands (via `@mediapipe/hands`)，实时手部追踪

**交互：**
- 进入足迹板块 → 弹出"打开摄像头体验手势导览？"的提示
- 用户允许 → 右上角显示小摄像头预览（200×150px），带手骨架线
- 手在摄像头画面里水平移动 → 地图视角水平平移（扫视城市）
- 握拳（5 根手指收拢）→ 手部位置下方最近城市激活 → 弹出该城市照片画廊
- 张开手掌 → 关闭画廊，继续浏览
- 关闭摄像头按钮 → 降级到鼠标模式

### 鼠标模式（授权拒绝或不支持）

- 鼠标悬停城市 → 城市光晕增强 + 周围粒子散开（用 Canvas/WebGL 粒子系统）
- 点击城市 → 弹出照片画廊
- 键盘 ← → 切换城市

### 粒子系统

- 每个城市是一个 glowing dot，周围有 20-50 个微粒子围绕
- 鼠标/手部光标接近 → 粒子被"推开"（reverse attraction）
- 离开 → 粒子回归

## 8. 技术栈增补

- **MediaPipe Hands**：`@mediapipe/hands` or `@mediapipe/tasks-vision` (新版)，用于手势追踪
- **粒子系统**：基于现有的 `@react-three/fiber`，用 `<Points>` 或 custom shader
- **动画**：现有 CSS + 新增 `framer-motion`（全屏翻页的 spring 动画）
- **SVG mask / CSS mask-image**：手电筒揭示效果，无需新库
